🔍 Поиск с подсветкой

JavaScript — это язык программирования, который используется для создания интерактивных веб-страниц.

JavaScript был создан в 1995 году и с тех пор стал одним из самых популярных языков.

Современный JavaScript используется не только в браузерах, но и на сервере (Node.js).

JavaScript поддерживает объектно-ориентированное и функциональное программирование.
