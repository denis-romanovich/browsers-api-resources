⚡ Скорость загрузки (приблизительно) — Mbps
⏱️ Задержка (RTT) — ms
💾 Режим экономии трафика —
⚠️ Важно: Значения являются приблизительными и могут не соответствовать реальной скорости интернета.
Браузер определяет effectiveType на основе текущей активности: slow-2g, 2g, 3g, 4g.
Точную скорость интернета API не показывает.

📋 Что означают значения?

  • slow-2g / 2g — очень медленное соединение (до 50 кбит/с)
  • 3g — среднее соединение (до 1.5 Мбит/с)
  • 4g — быстрое соединение (от 2 Мбит/с)
  • RTT — время отклика (чем меньше, тем лучше)
  • saveData — пользователь включил экономию трафика в браузере
